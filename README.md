# XMLUI Weather Dashboard

A simple weather dashboard built with XMLUI that displays current weather conditions for any city. It uses [wttr.in](https://wttr.in), a free weather service that requires no authentication or API key.

## Running the app

- Download and unzip this folder
- Run `start.sh` (Mac/Linux) or `start.bat` (Windows)
- The app will open in your browser at http://localhost:8000
